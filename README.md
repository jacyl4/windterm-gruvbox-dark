## Requirements:
* Font `GeistMono Nerd Font` [download link](https://github.com/ryanoasis/nerd-fonts/releases/download/v3.4.0/GeistMono.zip)

## Installation:
1. Move Folder `gruvbox` to WINDTERM_DIRECTORY\global\themes
2. Restart windterm
