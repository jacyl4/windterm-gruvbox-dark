## Requirements:
* Font `Cascadia Mono Light` [download link](https://github.com/ryanoasis/nerd-fonts/releases/download/v3.3.0/CascadiaMono.zip)

## Installation:
1. Move Folder `gruvbox` to WINDTERM_DIRECTORY\global\themes
2. Restart windterm
